# Epocrate-R-

**Epocrate-R-** is a starter Flutter application built with **Flutter** and **Dart**. This repository provides a basic mobile app structure that can be extended to build full-featured applications by adding custom screens, logic, and UI components.

## Features

- Basic Flutter project setup
- Organized folder structure
- Cross-platform support (Android & iOS)
- Ready for customization and further development

## Tech Stack

- Flutter
- Dart

## Getting Started

### Prerequisites

Ensure you have the following installed:

- Flutter SDK
- Android Studio or Visual Studio Code
- Android / iOS simulator or physical device
